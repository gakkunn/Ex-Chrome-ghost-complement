## Chrome Web Store Requirements

### Screenshots (Required)

- **Format**: PNG, JPG, GIF
- **Size**: 1280x800px
- **Count**: 1–5 recommended
- **Content**: Show key features of the extension
- **Filenames**: `screenshot-1-1280x800.png` … `screenshot-5-1280x800.png`

### Small Promo Tile (Recommended)

- **Size**: 440x280px (11:7)
- **Purpose**: Shown in recommendation sections and category listings
- **Filename**: `promo-small-440x280.png`

### Promo Image (Optional)

- **Size**: 1400x560px (5:2)
- **Purpose**: Promotional campaigns in the store
- **Filename**: `promo-1400x560.png`

### Tile Marquee (Optional)

- **Size**: 1400x560px (5:2)
- **Purpose**: Featured in Chrome Web Store carousel
- **Filename**: `tile-marquee-1400x560.png`
