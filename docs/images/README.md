## Chrome Web Store Requirements

### Screenshots (Required)

- **Format**: PNG, JPG, GIF
- **Size**: 1280x800px
- **Count**: 1-5 recommended
- **Content**: Show key features of the extension

### Promo Image

- **Size**: 1400x560px
- **Purpose**: Promotional campaigns in the store
- **Filename**: `promo-1400x560.png`

### Tile Marquee

- **Size**: 1400x560px
- **Purpose**: Featured in Chrome Web Store carousel
- **Filename**: `tile-marquee-1400x560.png`
